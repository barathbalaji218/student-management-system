# STUDENT MANAGEMENT SYSTEM (FLASK)

### Description:
A full-stack Flask-based Student Management System with role-based access, authentication, leave management, marks tracking, and production-grade deployment using Docker.

### Features :
1. Authentication(Admin and Student)
2. Student Dashboard
3. Admin Dashboard
4. Leave Request and Approval System
5. Marks and Profile Management
6. Secure Session and Role Protection
7. Dockerized Application 
8. Cloud Deployment 

### Tech Stack:
1. **Backend**:
    Flask(Python 3.13)
2. **Database**:
    MongoDB 
3. **Auth**:
    Session-based Auth
4. **User Interface**:
    Bootstrap 
5. **Container**:
    Docker
6. **Deployment**
    Render(Free Tire)
